# csz233.github.io

SmartpetFeeder project page for GitHub Pages.

How to publish:

1. Keep `index.html` in the repository root.
2. Keep the `SmartpetFeeder/` image folder in place.
3. In GitHub repository settings, set Pages source to the `main` branch and the root folder.

After that, the site can be served directly as a static page with no build step.
